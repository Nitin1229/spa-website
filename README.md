## Getting Started

1. Clone or download this repository
2. Open `index.html` in a web browser
3. No build process required - it's pure vanilla JavaScript!

## Form Features

The contact form includes:
- **Submit Button**: Validates and submits the form
- **Reset Button**: Clears all form fields
- **Preview Button**: Shows a preview of the form data
- **Save Draft Button**: Saves form data to browser's local storage

## Deployment to GitHub Pages

1. Push this repository to GitHub
2. Go to your repository settings
3. Navigate to "Pages" section
4. Select the branch (usually `main` or `master`)
5. Select the folder (usually `/root`)
6. Click "Save"
7. Your site will be available at `https://yourusername.github.io/repository-name`
